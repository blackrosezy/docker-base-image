###Generic Base Image##

Simple base image (inherited from Ubuntu 14.04 image) pre-installed with essential tools:
```
apt-transport-https
ca-certificates
curl
build-essential
git
htop
language-pack-en
less
make
python
python-dev
python-pip
rpl
software-properties-common
supervisor
syslog-ng-core
telnet
wget      
zip
```

Usage:
```
FROM morz/generic-base
```
